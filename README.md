# webpack-simple-react-typescript

Webpack 5 simple boilerplate for React 17 with Typescript 4, Jest & SASS

![Webpack simple](webpack-simple.jpg)

# Includes
- React 17
- React router 5
- Typescript 4
- Sass loader 
- Postcss loader
- Jest 27

# Instructions
- Clone this repo
- `npm install`
- `npm start`
- `npm run test:watch`
